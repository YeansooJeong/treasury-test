☁ Supabase 데이터 이전

현재 PC의 localStorage 데이터를 Supabase로 일괄 이전합니다. 중복 데이터는 자동으로 제외됩니다.

⏰ 자동 백업 설정

매 영업일(월~금) 오전 11시 59분에 자동으로 JSON 백업 파일을 다운로드합니다.
브라우저 탭이 열려있어야 동작합니다.

마지막 자동백업: —
↓ JSON 전체 백업

현재 Supabase에 저장된 모든 데이터를 JSON 파일로 내보냅니다. 정기적으로 백업하세요.

↑ JSON 데이터 복원

백업 JSON 파일을 업로드하여 Supabase 데이터를 복원합니다. 기존 데이터가 교체됩니다.
